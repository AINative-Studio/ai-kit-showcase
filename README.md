# AI Kit Showcase

Interactive demo site showcasing [AI Kit](https://github.com/AINative-Studio/ai-kit) capabilities.

## Features

- Streaming AI responses
- Agent orchestration demos
- Safety & moderation features
- Usage analytics dashboard

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This app is configured for Railway deployment:

1. Connect your GitHub repo to Railway
2. Railway will auto-detect Next.js and configure the build
3. Set any required environment variables in Railway dashboard
